## Allugator Front-end

Projeto front-end em Next.js para listar produtos disponíveis para aluguel, exibindo nome, imagem e valor. Permite busca por nome (completo ou parcial) e adicionar itens ao carrinho.

### Dependência do backend

Este front-end deve ser executado em conjunto com o backend disponível em:
https://github.com/LucasLisboaMotta/Allugator-Back-Nest

Certifique-se de subir o backend antes de iniciar o front-end.

### Como executar

1. Instale as dependências:

```
npm install
```

2. Inicie o servidor de desenvolvimento:

```
npm run dev
```

O projeto podera ser acessado pela rota http://localhost:3001

### Funcionalidades

- Listagem de produtos para aluguel
- Exibir detalhes de um produto
- Exibição de nome, imagem, valor e descrição
- Busca por nome (completo ou parcial)
- Adição de produtos ao carrinho
